## Supercademy - Getting started
First, run the development server:

```bash
npm run dev
# or
bun dev
```

If opened Storybook, run the following command:

```bash
npm run storybook
# or
bun storybook
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
